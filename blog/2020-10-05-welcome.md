こんにちは、サイトコアで仕事をしている原水です。このサイトは個人的に検証した内容をメモとしてまとめていきます。何を取り上げるかは手元で利用しているものとか、会社のお仕事で気づいたところ、仕事の内容のご紹介という感じになります。

<!--truncate-->

このサイトは以下のツールを利用して構築しています。

- **Docusaurus** : Facebook が提供する静的サイト生成ツールです。Markdown でコンテンツを書いて、ブログや記事になるので便利です。
- **Netlify** : Web サイトホスティングサービスとして Netlify を利用しています。個人でトラフィック少なければ無料で使えるので便利です。
- **GitHub** : コード管理ツールです。サイトのコードはすべて公開しています。
- **Visual Studio Code** : コードを書くツールとしては Visual Studio Code を利用しています。Docusaurus の設定の変更、Markdown の文書を書く、というだけなので、これだけで十分です。

ちなみに Docusaurus に関して、開発中となっている α版の v2 を利用しています。単にこちらの方が色々とできることが増えてくるだろう、ということと、何かあればフィードバックできるかな、というのがあります。合わせて、Netlify に関してのメモをこのサイトの最初のコンテンツとして展開していく予定です。

この記事は新しくなった時の最初の投稿であり、過去の技術的なブログで今も使えそうなものはここに随時移動していきます。
