昨年、このブログでも紹介していましたが Sitecore が買収した製品、Moosend を改めて Sitecore Send として企業向けのサービスとして展開を開始しました。今回はこの開始したばかりですがサービスの内容を紹介していきます。

## サービスのブランドとして

Moosend および Sitecore Send のサービスに関して提供をしているサービスは基本的には同じものとなります。以前は Moosend では機能を限定することで提供をしていた無料プランがありましたが、それは廃止されました。現在は、ほぼ全ての機能を利用することができる30日期間限定プランとう形となりました。

多くの機能がありますが、違いは以下のような形です（全てに備わっている機能は表からは外しています、詳しくは https://moosend.com/pricing/ でご確認ください）。

|  | 30日間無料 | Pro | Enterprise |
| 購読者数 | ○ | ○ | ○ |
| メールキャンペーン | ○ | ○ | ○ |
| トランザクションメール | - | ○ | ○ |
| ランディングページ | ○ | ○ | ○ |
| カスタムレポート | 1 | ○ | ○ |
| アカウントマネージャー | - | - | ○ |
| チームメンバー | 1 | 5 | 10+ |
| プライオリティサポート | - | - | ○ |
| SMTP Server | - | ○ | ○ |
| SSO & SAML | - | - | ○ |
| SLA | - | - | ○ |
| Dedicated IP | - | - |  |

これにより 30 日間の間に設定、評価をしていただき、そのまま導入につなげるということが可能となっています。

Moosend と Sitecore Send の違いは契約形態となります。基本的には Moosend はオンラインで申し込みをすることが可能となっており、支払いはクレジットカードを利用することでいつでも申し込みが可能となっています。Sitecor Send
