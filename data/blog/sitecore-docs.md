## 非公式サイト

### Sitecore クイックスタート

Sitecore 9.2 以降になりますが、インストールガイドほど詳細ではありませんが、Step by Step で手順を紹介しているクイックスタートというコンテンツを Github で展開をしています。

* [Sitecore クイックスタート](https://sitecorejapan.cmsdemo.jp/)

このガイドは 2020年10月現在、Sitecore 10.0 に関する情報を掲載しています。また、Sitecore Experience Accelerator および Sitecore JavaScript Service に関する情報を掲載しています。

上記のコンテンツに関しては、随時このサイトにマージしていって、最終的には廃止する形で考えています。すでに Sitecore JavaScript Services に関して紹介をするシリーズを開始しています。

* [Sitecore JSS シリーズ](/blog/sitecore-jss)

また Sitecore Experience Accelerator および Sitecore JavaScript Services に関しても掲載していきます。古くてメンテナンスされていませんがこちらも参考にしてください。

* [Sitecore Experience Accelerator](https://sitecorejapan.cmsdemo.jp/sxa/index.html)
* [Sitecore JavaScript Services](https://sitecorejapan.cmsdemo.jp/jss/index.html)

### サイトコア ユーザーガイド

Sitecore ユーザーガイドとして、Sitecore 9.3 のマニュアルを機械翻訳で展開しているサイトを立ち上げています。

* [Sitecore ユーザーガイド](https://doc.sitecorejp.net/)

バージョンごとの違いなどもありますが、使い方について参考にするサイトとしてご利用ください。
