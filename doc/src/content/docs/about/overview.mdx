## Sitecore Cloud Portal

Sitecore Cloud Portal は、Sitecore 製品を一元管理するためのポータルサイトです。ここでは、各種サービスの設定や管理、ユーザーの権限管理などを行うことができます。初めて Sitecore 製品を利用する際には、まずこのポータルにアクセスし、必要な設定を行うことが推奨されます。

- [詳しくはこちら](/cloud-portal/overview/)

## Engagement

### Sitecore CDP + Personalize

Sitecore CDP (Customer Data Platform) と Personalize は、顧客データの統合とパーソナライズされた体験の提供を支援するツールです。CDP は、様々なチャネルから収集された顧客データを統合し、分析するためのプラットフォームです。一方、Personalize は、そのデータを基にして、個々の顧客に最適化されたコンテンツやオファーを提供するためのツールです。これにより、顧客一人ひとりに対して、よりパーソナライズされた体験を提供することが可能になります。

これらのツールを活用することで、企業は顧客との関係を深め、より効果的なマーケティング活動を展開することができます。

- [詳しくはこちら](/cdp-personalize/overview/)

### Moosend / Send

Sitecore が提供するメールマーケティングツールは、顧客リストの管理、配信だけでなく、メール配信のシナリオを作ることができるオートメーションも含めて提供しており、強力なメールマーケティングツールです。利用規模にあわせて Moosend （中小企業向け）および Sitecore Send （大企業向け）を選択することができます。また開封確認やレポーティングも提供しており、メールのチャネルをより良いものにするための情報を得ることができます。

このツールを活用することで、メールのチャネルを通じたコミュニケーションの改善にご利用ください。

- [詳しくはこちら](/send/overview/)
