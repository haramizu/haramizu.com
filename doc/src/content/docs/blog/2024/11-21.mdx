## Moosend / Sitecore Send のコンテンツを追加しました

Sitecore が提供するメールマーケティングツールである Moosend および Sitecore Send に関するコンテンツを追加しました。

- [Moosend　/ Sitecore Send 概要](/send/overview/)

まだ Moosend の無料トライアルに関する記事のみですが、これまでブログで紹介していた Tips が随時追加される予定です。

## CDP + Personalize のコンテンツ提供の変更

初回に提供を始めた CDP + Personalize のコンテンツですが、要素を分けた方がわかりやすい形になるため、以下のように変更をしました。両方を紹介する以下のページには変更ありません。

- [Sitecore CDP + Personalize について](/cdp-personalize/overview/)

製品に関しての説明としては、以下のような構成とします。

- **Sitecore Persoanlize** : _/personalize_ のパスでコンテンツの展開をします
- **Sitecore CDP** : _/cdp_ のパスでコンテンツの展開をします

これに伴い、先行して提供していた Personalize のコンテンツの URL は変更となりました。
