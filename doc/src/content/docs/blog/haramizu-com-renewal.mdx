## Haramizu.com のリニューアルを始めました

これまで Haramizu.com では、仕事上で頂く質問や、このあたりを紹介しておくと便利かも、という記事を Tips という形で紹介をしてきましたが、記事が多くなりすぎてサイトでの可読性が悪くなっている状況でした。

そこで、新しいサイトを立ち上げて従来のトピックで今も有効なコンテンツをもう少し細かく紹介する形として、体系だって見れるようにしていきます。

第一弾として、以下の製品に関するブログの記事に関しては、以前に比べて見やすく参照できるようになったと思います。

- [Sitecore Cloud Portal](/cloud-portal/overview/)
- [Sitecore CDP + Personalize](/cdp-personalize/overview/)

またこのドキュメントサイトおよびブログに関しては、Astro を利用して作成しています。実際に作っている手順に関しては以下のページで参照できます。

- [Starlight の利用](/astro/starlight/)
- [Starlight にブログを追加する](/astro/starlight-blog/)

## 今後の方針として

基本的に新しい Tips に関してはこのサイトに掲載していく予定です。

とはいえ、これまで掲載していたブログに関しては、引き続き以下の URL で参照できます。

- https://blog.haramizu.com/ 

ある程度、このサイトにコンテンツの移行ができた段階で、blog 専用のサイトは終了として、トップレベルドメインでコンテンツを見れるようにする予定です。
